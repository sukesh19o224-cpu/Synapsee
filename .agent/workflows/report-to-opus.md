# Reporting to Opus

After Sonnet completes work, copy Sonnet's report to Opus chat.

## Quick Format

```
SONNET COMPLETED ORDER #[X]:
[Paste Sonnet's report here]
```

## If Sonnet is Blocked

```
SONNET BLOCKED ON ORDER #[X]:
[Paste the problem description]
```

Opus will then provide guidance or a solution.
